# Module 3 — Harris, FAST and Geometric Reasoning

This module contains tasks based on Harris/FAST corner detectors and camera-ground geometry.

## Programs

- `task_1.py` — project Harris points from one frame into the next frame using known motion.
- `task_2.py` — recover 3D ground points and estimate camera placement from them.
- `task_3.py` — compare Harris and FAST detections and visualize their overlap.
- `task_4.py` — compute 3D coordinates of Harris points on the road and test if they are coplanar.
- `task_5.py` — randomly sample Harris points until a planar road subset is found.
- `fast_detector.py` — standalone FAST detector demo.
