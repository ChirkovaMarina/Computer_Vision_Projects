# Module 2a — Morphological Operations

This module demonstrates thresholding and basic binary morphology for segmentation and structure extraction.

## Programs

- `1_threshold.py` — thresholding example on grayscale input.
- `2_morphologyEx.py` — examples of `cv.morphologyEx` operations.
- `3_hitmiss.py` — hit-or-miss transform example.
- `4_erod_and_dilate.py` — erosion and dilation examples.
- `Morphological_Operations_segmentation.py` — morphology-based segmentation pipeline for a larger scene.
