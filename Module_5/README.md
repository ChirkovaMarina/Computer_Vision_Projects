# Module 5 — Hough Transform and Projective Geometry

This module contains line- and circle-based tasks using the Hough transform, vanishing points, rail geometry and camera pitch estimation.

## Programs

- `task_1.py` — detect vertical lines, compute the 3D coordinates of their lower endpoints and place them on a local map.
- `task_2.py` — detect tram rails using the known track gauge.
- `task_3.py` — estimate the horizon line from rail-line intersections and a vanishing point.
- `task_4.py` — estimate the precise camera pitch from projected tram rails.
- `task_5.py` — detect traffic lights with Hough circles.
- `simple.py` — simple Hough transform implementation and visualization.
- `vanishing_point.py` — vanishing point estimation from line intersections.
