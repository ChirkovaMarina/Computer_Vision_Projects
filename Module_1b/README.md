# Module 1b — Camera Calibration

This module contains calibration examples based on checkerboards and circular calibration patterns, plus pose and motion visualization.

## Programs

- `calibration.py` — standard camera calibration with a checkerboard and undistortion preview.
- `calibration_fisheye.py` — fisheye calibration and undistortion using a checkerboard.
- `camera_motion.py` — estimates camera pose relative to the calibration board and visualizes motion in 3D.
- `Circles_calib.py` — camera calibration with an asymmetric circle grid.
