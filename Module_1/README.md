# Module 1 — Basic Image Operations

This module contains introductory OpenCV examples for loading, displaying, modifying and drawing on images.

## Programs

- `1_imread.py` — read and display an image.
- `2_cvtcolor.py` — convert between color spaces.
- `3_resize.py` — resize an image.
- `4_draw.py` — draw lines, rectangles, circles and text.
- `5_pyr_up_down.py` — use image pyramids for up/down scaling.
- `6_cv_merge.py` — split and merge BGR channels.
- `7_crop_flip.py` — crop and flip images.
- `8_copyMakeBorder.py` — add borders to an image.
- `9_arithmetic_operation.py` — arithmetic image operations.
- `10_bitwise_operation.py` — bitwise image operations and masking.
- `graph_draw.py` — render a mathematical graph into an image.
- `newtask__paint.py` — interactive mouse drawing example.
