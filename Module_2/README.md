# Module 2 — Image Preprocessing and Visibility Enhancement

This module contains classical image-processing scripts focused on visibility improvement for automotive scenes: glare reduction, sharpening, color processing and simple frame-based correction.

## Programs

- `Filters.py` — examples of blur and convolution filters.
- `ColorSpaces.py` — color space conversions and channel manipulations.
- `Translations.py` — affine translation examples.
- `Rotations.py` — image rotation examples.
- `sharpened.py` — simple sharpening for video frames.
- `Sun_blicks.py` — reduce sun glare on glass.
- `detection_flares.py` — glare detection helpers based on HLS.
- `reduce_glare.py` — polynomial and gamma-based glare reduction pipeline.
- `task_1.py` — reduce headlight glare using a distance-transform-based mask.
- `task_2.py` — reduce overexposure caused by a snow storm.
- `task_3.py` — suppress wipers and rain drops by frame differencing.
- `task_4.py` — reduce direct sunlight artifacts in the upper camera area.
- `task_5.py` — reduce glare on wet asphalt.
- `task_5(right).py` — reduce glare from oncoming headlights.
- `task_7.py` — another direct-sun overexposure correction task.
