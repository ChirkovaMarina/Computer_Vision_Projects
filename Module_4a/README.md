# Module 4a — Feature Points, Matching and Motion

This module contains keypoint-based tasks using SIFT or ORB, plus geometric filtering and motion analysis.

## Programs

- `task1.py` — SIFT keypoint matching between frames, ground filtering and 3D displacement estimation.
- `task2.py` — SIFT matching with homography estimation and RANSAC outlier rejection.
- `task3.py` — decide whether the observed object is moving based on keypoint displacement.
- `task5.py` — detect moving objects while the tram itself is stationary.
